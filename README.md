# Neumorphism Loading Spinner in HTML & CSS | CSS Animations

Um projeto simples de **loading spinner com efeito neumorphism** usando HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar um **spinner de carregamento com estilo neumorphism**, utilizando sombras suaves e animações CSS.

## 🛠️ Tecnologias

- HTML
- CSS (animation, box-shadow)

## 🎯 Funcionalidades

- Spinner animado
- Estilo neumorphism
- Design moderno e elegante

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
